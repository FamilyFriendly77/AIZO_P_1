randomSeed - random data generation seed 
inTestMode - true/false indicates if program should only test testSortingAlg and print results to the screen (sets printAfterGenerating and printAfterSorting to true)
testFilename - name of the file in executable directory that contains data that testSortingAlg should be tested on.
testCasesArrayLengths - an array of lengths that progran should generate
testRepetitionCounter - how many arrays of each length to generate and sort
testSortingAlg - name of the sortingAlg to be tested - heapSort, quickSort, insertSort, shellSortTwo, shellSortTwo
printAfterGenerating - if generated/read from file array should be printed to the console
printAfterSorting - if sorted array should be printed to the console
quickSortPivot - dictates how pivot should be choosen - FIRST, LAST, MIDDLE, RANDOM
testIfSorted - if success of sorting should be tested
test[INT/DOUBLE/FLOAT/CHAR] - if tests should be run for specified datatype
